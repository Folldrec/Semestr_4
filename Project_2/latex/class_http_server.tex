\doxysubsection{Detailed Description}
HTTP/1.\+1 сервер на сирих сокетах. 

Кожен клієнт обробляється у окремому std\+::thread. Реєстрація маршрутів через метод \doxylink{class_http_server_a3eaec2fea369bc17e06ee8eb4cf241ac}{route()}.

\begin{DoxyNote}{Note}
Частина патерну Facade — приховує деталі сокетів від api\+\_\+handler. 
\end{DoxyNote}

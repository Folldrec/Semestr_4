\doxysubsection{Detailed Description}
Гра у бібліотеці Steam гравця. 

Зберігає час гри, жанри та досягнення. Поле {\ttfamily excluded} дозволяє м\textquotesingle{}яко видалити гру з підрахунку без втрати даних (патерн Command).

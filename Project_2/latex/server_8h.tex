\doxysubsection{Detailed Description}
Простий кросплатформний HTTP/1.\+1 сервер. 

Реалізований на сирих сокетах (Winsock2 на Windows, POSIX на Linux/mac\+OS). Підтримує реєстрацію маршрутів та обробку запитів у окремих потоках.

Detailed Description

Простий кросплатформний HTTP/1.1 сервер.

Реалізований на сирих сокетах (Winsock2 на Windows, POSIX на Linux/macOS). Підтримує реєстрацію маршрутів та обробку запитів у окремих потоках.

Використання:
HttpServer server(8080);
server.route("GET", "/api/hello", [](const HttpRequest& req) {
return makeJsonResponse("{"ok":true}");
});
server.run(); // блокуючий виклик

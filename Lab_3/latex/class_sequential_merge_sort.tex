\doxysubsection{Detailed Description}
Classic sequential merge sort. 

Time complexity\+: O(n log n) Space complexity\+: O(n)

Runs entirely on the calling thread — no parallelism. Used as the baseline for benchmarks and as the fallback inside \doxylink{class_parallel_merge_sort}{Parallel\+Merge\+Sort} for small sub-\/arrays.

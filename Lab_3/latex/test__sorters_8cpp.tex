\doxysubsection{Detailed Description}
Unit tests for \doxylink{class_sequential_merge_sort}{Sequential\+Merge\+Sort} and \doxylink{class_parallel_merge_sort}{Parallel\+Merge\+Sort}. 

Uses a lightweight hand-\/rolled test framework so no external dependency (Google\+Test, Catch2, …) is required.

Covers\+:
\begin{DoxyItemize}
\item Correct sorting\+: typical, empty, single element, two elements, already-\/sorted, reverse-\/sorted, duplicates, all-\/equal, negatives, mixed positive/negative, large random input.
\item Input not mutated.
\item Sequential and parallel results are identical.
\item Invalid constructor arguments throw std\+::invalid\+\_\+argument. 
\end{DoxyItemize}

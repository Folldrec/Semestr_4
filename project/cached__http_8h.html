Detailed Description

Патерн Decorator — кешуючий HTTP клієнт.

Обгортає будь-яку реалізацію IHttp та додає кешування відповідей в пам'яті. При повторному запиті того самого URL повертає збережений результат без звернення до мережі.

Патерн: Decorator (Структурний)
Переваги: прискорює завантаження жанрів при порівнянні гравців
Приклад:
auto http = std::make_shared<CachedHttp>(
std::make_shared<BasicHttp>(), 500);
auto resp = http->get("https://store.steampowered.com/...");
std::cout << "Hits: " << http->hits(); // кількість cache hits
